Shared Cursors

A Node.js application I made which allows you to see the cursor position of everyone else connected to the page through utilization of websockets and sockets.io. Currently also supports setting a name for yourself and displaying chat messages, although your own messages will not be displayed over your own cursor. Due to Heroku not currently supporting websockets, this app uses the xhr long-polling option that sockets.io provides. I recommend you open multiple windows to try it out.

Demonstration Github

Code-Box

A web-app I made a while back when I was first learning about HTML, CSS, and Javascript, and I wanted to play around with the Dropbox API. I utilized the Flask python framework and extensively used AJAX calls. This app basically acts as an in-browser text editor for the files in your Dropbox. Once you login, the left side displays folders and files and you can select a file to start editing it and then click the top right save button to save.

Demonstration Github

Techcrunch News Reader

Made this app for the Pulse Coding Challenge over one day. Reads the RSS feed of TechCrunch and gets the four most recent articles. Upon clicking on an article, the user gets a full view of the article as well as the ability to submit comments on the article which are saved in for other users to see.

Demonstration Github

jQuery Carousel Plugin

jQuery plugin which makes a vertical infinite carousel of an HTML unordered list of elements. A user can keep scrolling through a list of elements but they will keep repeating, thus creating the effect of looking like a spinning wheel. Also has the option of automatically scrolling down one list element over a set interval.
